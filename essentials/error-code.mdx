## HTTP Status

| HTTP Status | 涵義                      |
| ----------- | ------------------------- |
| 401         | Unauthorized              |
| 404         | Resource Not Found        |
| 422         | Wrong data in request     |
| 500         | Server internal error     |
| 503         | Service Unavailable       |
