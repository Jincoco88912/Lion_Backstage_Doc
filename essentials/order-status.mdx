## 1. 新夥伴串接請看此

訂單狀態如下表：

| 狀態值 | 意義                 | 子狀態 |
| ------ | -------------------- | ------ |
| -1     | 初始訂單(訂單創建成功) | null   |
| 0      | 等待供應商確認        | null   |
| 1      | 訂單完成(供應商已確認) | null   |
| 2      | 訂單失敗              | 參見下方子狀態說明 |
| 3      | 訂單已取消            | null   |

### 1.1 子狀態說明

- 訂單失敗的子狀態:
  1. 下訂失敗：無法查詢到最新價格
  2. 訂單已過期：未在 10 分鐘內完成下訂
  3. 下訂失敗：供應商回應不預期狀態值
  4. 下訂失敗：產品發生變價，成本變貴
